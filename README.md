# Mytheresa Product Scraper 🥿🛍️

This Python script uses **Selenium** and **BeautifulSoup** to scrape product information from the Mytheresa men's shoes page.

## 🔍 Features

- Scrolls the entire page to load all products
- Extracts:
  - Brand
  - Product Name
  - Offer Price
  - Listing Price
  - Discount
  - Sizes Available
  - Image URL
